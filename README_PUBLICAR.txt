3. O site abre no navegador.

COMO PUBLICAR DE GRAÇA NO CLOUDFLARE PAGES
1. Crie uma conta gratuita na Cloudflare.
2. Entre em Workers & Pages > Create application > Pages.
3. Escolha Direct Upload.
4. Faça upload desta pasta ou do ZIP descompactado.
5. Depois conecte o domínio arquitetorenan.com em Custom domains.

COMO ATUALIZAR DEPOIS
- Para alterar textos: edite index.html.
- Para alterar cores e espaçamentos: edite styles.css.
- Para trocar imagens: substitua os arquivos em assets/img/ mantendo o mesmo nome.

OBSERVAÇÃO
Este site não usa Wix, WordPress, plugins pagos, banco de dados ou ferramentas externas. É uma página estática, leve e barata para manter.
